# <img width="50" height="50" alt="ddfsdsfsdffsddsf" src="https://github.com/user-attachments/assets/fd4dea43-75c5-4a72-b638-ace39e3282a3" /> ObhodBApp
**ObhodBApp** - Графическая оболочка для Clash-meta (Mihomo), написанная на avalonia. Вдохновленна клиентом OpenVPN Connect. Программа запускает Clash-meta, форматирует все его логи, собирает информацию использования сети и может самостоятельно автоматически запускаться при запуске системы.

> [!WARNING]  
> Программа содержит большое колличество багов и недоработок. Если у вас возникают ошибки - напишите их в [Issues](https://github.com/G0Shark/ObhodBApp/issues).

---

## Установка (Windows)
1. [Скачайте последний релиз](https://github.com/G0Shark/ObhodBlokirovok/releases/latest)  
2. Разархивируйте архив в удобную папку  
3. Запустите `ObhodBApp.exe`
